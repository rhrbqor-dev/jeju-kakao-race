등록된 정답 설명 이미지 없음

제한: 정답 설명 이미지 최대 5장, 이미지 1장당 최대 2MB, JPG/PNG/WEBP만 업로드할 수 있습니다. 정답을 맞힌 뒤 설명과 함께 표시됩니다.

예: -5로 입력하면 오답을 1회 입력할 때마다 팀 점수에서 5점이 차감됩니다.

예: -10으로 입력하면 해당 미션에서 힌트를 처음 사용할 때 팀 점수에서 10점이 차감됩니다.

사진 자동 승인 설정

사진 미션을 운영자가 직접 승인하지 않고 자동으로 승인할지 설정합니다. 자동 승인 ON + 시간 설정 OFF이면 항상 자동 승인됩니다. 시간 설정 ON이면 입력한 시간 안에 업로드된 사진만 자동 승인됩니다.

수료증 설정

완주한 참가자에게 보여줄 수료증 화면을 설정합니다. 배경 이미지를 업로드하고 문구를 입력하면 완주 후 카카오톡에 “수료증 보기” 버튼이 표시됩니다.

등록된 배경 이미지 없음

제한: 배경 이미지 1장, 최대 5MB, JPG/PNG/WEBP만 업로드할 수 있습니다. 권장 비율은 세로형 A4 또는 720×1024 비율입니다.

권장: 팀명과 참가자 이름을 함께 넣어주세요. 줄바꿈은 Enter로 입력합니다. 예: {team_name} 다음 줄 {member_name} 님

수료증 위치 / 폰트 크기 설정

좌표 기준은 720×1040입니다. 각 문구는 지정한 박스 안에서만 표시되며, 글자가 길면 최대 폰트에서 최소 폰트까지 자동으로 줄어듭니다.
